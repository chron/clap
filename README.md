### Setting up a dev environment

```
yarn install
npm i -g netlify-cli
netlify dev
```

### Environment variables you will need

* PUSHER_APP_ID
* PUSHER_CLUSTER
* PUSHER_KEY
* PUSHER_SECRET
* FAUNADB_SECRET_KEY
